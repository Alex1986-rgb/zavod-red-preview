## 1. Адаптив и отображение на всех устройствах
- Перепроверены **все шаблоны** на 320 / 375 / 768 px — **0 горизонтальных переполнений**.
- Тёмная/светлая тема; широкие таблицы скроллятся внутри контейнера (не ломают страницу).
- Исправлено «дёрганье» элементов при наведении на мобильном.

## 2. Шапка и навигация
- **Фикс кнопки «Оставить заявку»**, которая уезжала за экран на MacBook: бургер-меню ≤1240px,
  быстрые ссылки видны на десктопе, телефоны в шапке только на широких экранах (≥1681px).
- Мобильная липкая панель: убрана «Позвонить», осталась «Получить расчёт» на всю ширину.

## 3. Формы и конверсия
- **Синхронизация форм с бэкендом:** форма импортозамещения (`#impForm`) была «мёртвой» (без `name`
  и обработчика) — добавлен универсальный обработчик `.lead-form` → заявка уходит на `api/feedback.php`
  с UTM/referrer + цель Метрики `zayavka`. Телефон валидируется по числу цифр.
- В CTA-блоках кнопки-телефоны заменены на контекстные (Подбор / Бренды / Каталог / Оставить заявку).
- Убран дублирующий CTA-блок внизу главной.

## 4. SEO — техническая оптимизация
- **Title:** все ≤60 символов, уникальны (укорочено 76 длинных).
- **Description:** все 150–160, уникальны (укорочено 85 длинных).
- **Нижний SEO-текст** на 99 страницах: +2 тематические таблицы и +2 абзаца, свёрнуты под стрелку
  «Подробнее» (виден только лид-абзац).
- H1 (по одному), `alt` у картинок, canonical, Open Graph, schema.org — в норме на всех страницах.

## 5. Калькулятор подбора редуктора (ключевое)
- Обработана присланная **«Таблица подбора.xlsx»** (8 217 строк). Excel испортил часть чисел в даты —
  **восстановлено точно**: обороты/передаточное — по точным соотношениям (n_вых = n_вх / U), мощность —
  снапнута к стандартному ряду двигателей, момент — 99.5% из таблицы + 41 строка по КПД самой модели.
- **Объединённая база:** 30 курируемых моделей (с ГОСТ-обозначениями и импортными аналогами SEW/NORD/
  Motovario) + 104 модели из таблицы = **134 модели, 8 680 строк**.
- Калькулятор работает **на странице подбора** (все типы) и **встроен в 6 страниц каталога**,
  залоченным на свой тип (червячные → только червячные и т.д.).
- **SEO калькуляторов:** заголовки «Калькулятор подбора … по параметрам», якорь `#kalkulyator`,
  schema.org **HowTo** на 7 страницах, перелинковка, оптимизирован title страницы подбора.

## 6. Контент: 20 статей + кейсы
- **20 новых статей блога** под калькулятор (через скилл `zavod-blog-article` + агенты):
  подбор по мощности/моменту/оборотам/передаточному; по типам редукторов; калькулятор/таблица/
  типоразмер/EVL; расчёт момента/консольная нагрузка/частотник/нагрузка; аналог по передаточному/
  шильдик/ошибки/1500 об-мин. Каждая: title≤60, desc 150–160, schema Article+Breadcrumb+FAQ,
  4–5 таблиц, FAQ, перелинковка на калькулятор.
- **2 новых кейса:** подбор через калькулятор; замена импорта по передаточному.

## 7. Брендбук и документация
