# Брендбук — Завод Редукторов

Краткое руководство по фирменному стилю сайта zavod-red.ru. Источник истины для
цветов и компонентов — переменные в `assets/inner.css` (`:root`).

---

## 1. Бренд

- **Название:** Завод Редукторов
- **Юр. лицо:** ООО «НИИ АТТ» (ИНН 7452136680), Челябинск
- **Суть:** производитель промышленных редукторов и мотор-редукторов; подбор,
  импортозамещение приводов, поставка по России.
- **Позиционирование:** «от производителя» — цена без посредников, инженерный
  подбор по задаче, замена импорта без переделки оборудования.
- **Tone of voice:** по-инженерному конкретно и спокойно, без маркетингового шума.
  Говорим про задачу клиента (момент, обороты, присоединительные размеры, сроки),
  а не «купите сейчас». Цены на импорт — всегда «по запросу», не публикуем.

---

## 2. Цвета

Палитра тёмная, промышленная, с фирменным красным акцентом.

| Роль | Переменная | HEX | Назначение |
|---|---|---|---|
| Фон | `--bg` | `#0b151d` | основной фон страниц |
| Фон 2 | `--bg2` | `#0e1a24` | секции-чередование |
| Карточка | `--card` | `#14222e` | блоки, карточки |
| Карточка 2 | `--card2` | `#172836` | вложенные/hover |
| Линия | `--line` | `#22333f` | границы, разделители |
| **Акцент (бренд)** | `--red` | **`#e11b1b`** | кнопки, ссылки-акценты, логотип |
| Акцент 2 | `--red2` | `#ff3b30` | hover/иконки акцента |
| Доп. синий | `--blue` | `#2f7fc2` | вторичные акценты |
| Сталь | `--steel` | `#7fa8c4` | иконки/детали |
| Текст | `--text` | `#e9eff4` | основной текст на тёмном |
| Приглушённый | `--muted` | `#94a8b5` | подписи, вторичный текст |
| Чернила | `--ink` | `#13212c` | текст на светлом (светлая тема) |

**Правило акцента:** красный `#e11b1b` — только для главных действий и акцентов
(CTA, активные пункты, выделение ключевых слов). Не заливать им крупные плоскости.

**Светлая тема** включается переключателем (`data-theme="light"`), цвета берутся из
тех же переменных; логотип автоматически меняется на тёмный вариант.

---

## 3. Типографика

- **Шрифт:** `--font: 'Segoe UI', -apple-system, BlinkMacSystemFont, Roboto,
  'Helvetica Neue', Arial, sans-serif` (системный, без веб-загрузки — быстрая отдача).
- **Заголовки:** жирные (700–800), плотный межбуквенный, ключевое слово — красным.
- **Текст:** 15–16px, межстрочный 1.6–1.75; вторичный — цветом `--muted`.
- **Кнопки/подписи:** 14–15px, 600–700.

---

## 4. Логотип

- Файлы: `assets/zr_logo.webp` (светлый — для тёмного фона),
  `assets/zr_logo_dark.webp` (тёмный — для светлой темы).
- Не сжимать по ширине (`flex-shrink:0`), сохранять пропорции; высота в шапке
  ~44px (десктоп) → 36–40px (мобайл).
- Охранное поле — не менее высоты буквы логотипа со всех сторон.

---

## 5. UI-компоненты

- **Кнопки:** `.btn` (основная — красная `--red`), `.btn.ghost` (контурная),
  `.btn.lg` (крупная), `.btn.sm` (компактная для шапки). Скругление `--r: 14px`.
- **Карточки:** фон `--card`, граница `--line`, радиус 14px, мягкая тень.
- **Сетка/контейнер:** макс. ширина `--maxw: 1240px`, центрирование, боковые отступы.
- **Шапка:** sticky, blur-подложка; горизонтальное меню ≥1241px, бургер ≤1240px;
  телефоны в шапке — только ≥1681px. Кнопка «Оставить заявку» видна всегда (кроме
  ≤430px, где её заменяет липкая нижняя панель «Получить расчёт»).
- **Формы заявки:** модалка (`data-zayavka`) и встроенные `.lead-form`; обязательны
  honeypot и согласие на обработку данных.

---

## 6. Радиусы, тени, отступы

- Радиус по умолчанию: **14px** (`--r`).
- Секции: вертикальный ритм ~52–80px (уменьшается на мобайле).
- Тени мягкие, тёмные (`rgba(0,0,0,.4–.5)`), без резких контуров.

---

## 7. Что нельзя

- Публиковать цены на импортные редукторы (только «по запросу»).
- Заливать красным крупные области или использовать его для обычного текста.
- Растягивать/искажать логотип, ставить светлый логотип на светлый фон.
- Дублировать `<title>` между страницами; держать `<title>` ≤60 символов.
