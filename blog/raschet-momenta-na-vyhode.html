Расчёт крутящего момента на выходном валу редуктора

Опыт инженеров Завода Редукторов · время чтения ~6 минут

Крутящий момент на выходном валу — это главный силовой параметр, по которому подбирают типоразмер редуктора. Его считают по простой формуле через мощность, КПД и выходные обороты. Разберём формулу, покажем три примера расчёта и объясним, почему без учёта КПД и сервис-фактора подбор приводит к перегрузу.

Основная формула расчёта момента

Момент на выходном валу редуктора связывает мощность двигателя и обороты рабочего органа. Базовая инженерная формула:

T2 = 9550 · P · η / n2
  • T2 — крутящий момент на выходном валу, Н·м;
  • P — мощность двигателя, кВт;
  • η — КПД редуктора (доля единицы);
  • n2 — обороты выходного вала, об/мин.

Коэффициент 9550 возникает из перевода единиц: момент в Н·м равен 9550·кВт/об·мин−1 (точное значение 60000/2π ≈ 9549). Эта формула — рабочий инструмент при подборе по мощности, когда момент на машине напрямую не задан.

Выходные обороты, в свою очередь, получают из входных через передаточное число: n2 = n1 / i. Сначала считают n2, затем подставляют в формулу момента.

Пример 1. Цилиндрический редуктор

Двигатель P = 4 кВт, синхронные обороты n1 = 1500 об/мин, передаточное число i = 32, КПД цилиндрического редуктора η = 0,96.

  • Выходные обороты: n2 = 1500 / 32 ≈ 47 об/мин;
  • Момент: T2 = 9550 · 4 · 0,96 / 47 ≈ 780 Н·м.

Это рабочий момент. Для подбора его умножают на сервис-фактор и сравнивают с номинальным моментом типоразмера по каталогу.

Пример 2. Червячный редуктор

Тот же двигатель 4 кВт и i = 40, но КПД червячного редуктора при таком передаточном числе ниже — η ≈ 0,75.

  • n2 = 1500 / 40 = 37,5 об/мин;
  • T2 = 9550 · 4 · 0,75 / 37,5 ≈ 764 Н·м.

Обратите внимание: КПД 0,75 против 0,96 «съедает» около четверти потенциального момента. Игнорировать его нельзя — иначе расчётный момент окажется завышен, а реальная отдача меньше.

Пример 3. Через входной момент и передаточное число

Если известен момент на входном валу T1, выходной момент считают напрямую:

T2 = T1 · i · η

Пусть T1 = 25 Н·м, i = 31,5, η = 0,97. Тогда T2 = 25 · 31,5 · 0,97 ≈ 764 Н·м. Передаточное число умножает момент во столько же раз, во сколько снижает обороты, за вычетом потерь на КПД.

КПД по типам редукторов

КПД сильно зависит от типа передачи и передаточного числа. Ориентировочные значения для одной ступени:

Цилиндрический (1 ступень)0,97–0,98
Цилиндрический (2–3 ступени)0,94–0,97
Коническо-цилиндрический0,92–0,96
Планетарный0,95–0,98
Червячный (i до 20)0,80–0,90
Червячный (i 40–80)0,50–0,75

Запас по моменту и сервис-фактор

Расчётный момент — это не итог подбора, а только его начало. Номинальный момент типоразмера должен превышать рабочий с запасом:

T2ном ≥ T2рабочий · Sf

Сервис-фактор Sf учитывает характер нагрузки, число пусков в час и продолжительность работы. Подробнее о его выборе — в статье «Сервис-фактор редуктора» и материале «Подбор редуктора по характеру нагрузки». Ниже — типовые значения для расчёта.

Характер нагрузкиСервис-фактор Sf
Равномерная, S1, редкие пуски1,0–1,25
С умеренными толчками1,25–1,5
Ударная, частые пуски/реверсы1,5–2,0 и выше

Например, рабочий момент 780 Н·м при ударной нагрузке с Sf = 1,8 требует редуктора с T2ном ≥ 1404 Н·м — то есть на типоразмер крупнее, чем по «голому» расчёту.

Вывод

Момент на выходном валу считается по формуле T2 = 9550·P·η/n2, обязательно с учётом КПД конкретного типа редуктора. Полученное значение умножают на сервис-фактор и сравнивают с каталожным T2ном. Чтобы не считать вручную, откройте калькулятор подбора редуктора — он рассчитает обороты, момент и предложит типоразмер. Для проверки под реальные условия пришлите инженеру режим работы и нагрузки на вал.

Вопросы и ответы

Частые вопросы

По какой формуле считают крутящий момент на выходном валу редуктора?
Базовая формула: T2 = 9550 · P · η / n2, где T2 — момент на выходном валу (Н·м), P — мощность двигателя (кВт), η — КПД редуктора (доля), n2 — обороты выходного вала (об/мин). Коэффициент 9550 переводит кВт и об/мин в ньютон-метры. Например, при P=4 кВт, η=0,95 и n2=47 об/мин момент равен T2 = 9550·4·0,95/47 ≈ 772 Н·м.
Зачем в формуле момента учитывать КПД редуктора?
КПД показывает, какая доля мощности доходит до выходного вала после потерь в зацеплении и подшипниках. У цилиндрических редукторов η ≈ 0,96–0,98, у червячных при больших передаточных числах он падает до 0,5–0,7. Если не учесть КПД, расчётный момент червячного редуктора окажется завышенным, а реальная нагрузочная способность — ниже ожидаемой.
Как пересчитать момент через передаточное число?
Момент на выходе можно выразить через входной момент: T2 = T1 · i · η, где T1 — момент на входном валу, i — передаточное число, η — КПД. Поскольку n2 = n1 / i, эта запись эквивалентна формуле через мощность. Передаточное число умножает момент во столько же раз, во сколько снижает обороты, минус потери на КПД.
Чем отличается номинальный момент редуктора от расчётного?
Расчётный (рабочий) момент — это то, что нужно вашему механизму. Номинальный момент редуктора (T2ном по каталогу) — максимально допустимый для длительной работы. Подбор верен, когда T2ном ≥ T2рабочий · Sf, где Sf — сервис-фактор. Запас по моменту покрывает пуски, удары и реверсы.
Какой момент брать для подбора, если известна только мощность?
Если момент на машине напрямую не задан, его вычисляют из мощности и требуемых выходных оборотов по формуле T2 = 9550 · P · η / n2. Достаточно знать мощность двигателя, КПД редуктора по типу и обороты, которые нужны на рабочем органе. Полученное значение и есть исходный момент для подбора типоразмера.
Как момент связан с радиальной нагрузкой на вал?
Если на выходном валу стоит звёздочка, шкив или шестерня радиусом R, то окружное усилие F = T2 / R создаёт радиальную (консольную) нагрузку на вал. Поэтому после расчёта момента отдельно проверяют допустимую консольную нагрузку редуктора — она может ограничить выбор раньше, чем сам момент.
Можно ли рассчитать момент онлайн?
Да. Введите мощность, обороты и тип редуктора в калькулятор подбора на странице «Подбор» — он рассчитает выходные обороты, момент с учётом КПД и предложит подходящий типоразмер. Для точного подбора пришлите инженеру условия работы: режим, число пусков, нагрузки на вал.

Чтобы выполнить расчёт крутящего момента на выходном валу редуктора, используют формулу T2 = 9550·P·η/n2: мощность двигателя в кВт, КПД по типу передачи и выходные обороты дают момент в ньютон-метрах. Полученное значение с сервис-фактором сравнивают с номинальным моментом типоразмера.

Подробнее: характеристики, таблицы и полезные ссылки

Расчёт момента на выходе — отправная точка подбора редуктора по силовому параметру. Сначала по входным и требуемым выходным оборотам находят передаточное число i = n1/n2, затем считают момент с учётом КПД конкретной передачи. Червячные редукторы при больших передаточных числах теряют до половины мощности, поэтому их момент всегда считают с реальным, а не идеальным КПД.

После расчёта рабочий момент умножают на сервис-фактор и подбирают типоразмер так, чтобы каталожный номинальный момент имел запас. Отдельно проверяют консольную нагрузку на вал: окружное усилие от звёздочки или шкива создаёт радиальную силу, которая может ограничить выбор раньше момента. Импортные приводы подбираем как габаритные аналоги, цены на импорт — по запросу.
