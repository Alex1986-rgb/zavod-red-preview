# Завод Редукторов — сайт zavod-red.ru

Корпоративный сайт производителя промышленных **редукторов и мотор-редукторов**
ООО «НИИ АТТ» (Челябинск). Задача сайта — приводить заявки на подбор/поставку
редукторов и импортозамещение приводов SEW, NORD, Bonfiglioli и др.

**Стек:** статический HTML + CSS + ванильный JS, без сборки и зависимостей.
Хостинг — обычный (GitHub Pages для превью / shared-хостинг с привязкой к домену).
Заявки с форм уходят на PHP-бэкенд CRM (отдельный репозиторий `zavod-red-crm`).

## Ключевые возможности

- **Адаптивность** 320–1920px. Шапка: горизонтальное меню на широких экранах,
  бургер ≤1240px; контент без горизонтального скролла, широкие таблицы скроллятся
  в контейнере.
- **Тёмная/светлая тема** — переключатель в шапке, выбор сохраняется в `localStorage`
  (логотип меняется на тёмный вариант для светлой темы).
- **SEO**: уникальные `<title>` (≤60 симв.) и `meta description` на каждой странице,
  микроразметка schema.org (`Organization`, `Product`, `FAQPage`, `BreadcrumbList`,
  `Article`), хлебные крошки, человекочитаемые URL, `sitemap.xml`, `robots.txt`.
- **Конверсия**: модальная форма заявки (`data-zayavka`), форма подбора по шильду,
  захват UTM/referrer/page_url (first-touch в `localStorage`), цель Яндекс.Метрики
  `reachGoal('zayavka')`, honeypot от спама.
- **Лиды** уходят на `api/feedback.php` (CRM) → MySQL + Telegram + e-mail.

## Локальный запуск

Это статика — достаточно любого файлового сервера:

```bash
cd zavod_preview
python3 -m http.server 4137
# открыть http://127.0.0.1:4137/index.html
```

## Боевая сборка / деплой

Перед заливкой на хостинг прогоняется prod-трансформация: открытие `robots.txt`,
вставка счётчика Яндекс.Метрики в `<head>`, добавление `api/` (обработчик формы).
Готовый архив заливается в корень сайта рядом с бэкендом CRM.

При правках CSS/JS поднимайте версию в ссылках (`inner.css?v=N`, `modal.js?v=N`),
чтобы сбросить кэш у браузеров.
